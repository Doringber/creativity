# 🎯 AR Catch — משחק AR בסגנון פוקימון גו

משחק וובי (Web/PWA) שרץ ישירות בדפדפן של כל מכשיר אנדרואיד (וגם iOS / דסקטופ).
ברגע שמפעילים את המצלמה, **דמויות ומטבעות מתחילים לקפוץ מסביבך** ואתה צריך לתפוס
אותם בלחיצה לפני שהזמן נגמר — עם ספירת תפיסות, קומבואים, שלבים ולוח תוצאות מגניב.

> נבנה כ-Web/PWA במכוון: אין צורך בהתקנה מחנות, רץ מיד דרך הדפדפן, וניתן
> "להתקין" כאפליקציה למסך הבית בלחיצה אחת.

## ✨ מה יש במשחק

- **AR מבוסס מצלמה** — וידאו חי מהמצלמה האחורית כרקע, עם דמויות מונפשות מעליו.
- **דמויות שקופצות** 👾🐲👽 — נדלקות במקומות אקראיים על המסך, קופצות, ונעלמות אם לא תפסת בזמן.
- **מטבעות** 🪙 — מסתובבים, שווים נקודות.
- **יצורים נדירים** ⭐💎 — נדירים, נוצצים, ושווים פי הרבה (50 נק׳).
- **פצצות** 💣 — היזהר! לחיצה עליהן מורידה נקודות ומאפסת קומבו.
- **קומבו** 🔥 — תפיסות רצופות מכפילות נקודות (עד x9).
- **שלבים** 🚀 — ככל שמתקדמים, הדמויות צצות מהר יותר.
- **לוח תוצאות** 🏅 — Top 10 עם מדליות, שמירת שם, שיא אישי (נשמר במכשיר).
- **משוב מלא** — צלילים (Web Audio, בלי קבצים), רטט (Vibration API), חלקיקים ואפקטים.
- **PWA** — ניתן להתקנה למסך הבית ועובד גם אופליין (Service Worker).
- **ממשק בעברית RTL** ומותאם למובייל (מסך מלא, safe-area).

## 🚀 איך מריצים

המשחק הוא קבצים סטטיים בלבד. צריך להגיש אותם דרך HTTP (המצלמה דורשת
`https://` או `localhost` — לא עובד מ-`file://`).

### מקומית
```bash
# מתוך תיקיית הפרויקט
python3 -m http.server 8000
# ואז לפתוח בדפדפן:  http://localhost:8000
```

### בטלפון אנדרואיד (הדרך הכי כיפית)
1. פרסו את התיקייה לכל אחסון סטטי עם **HTTPS** (GitHub Pages / Netlify / Vercel / Cloudflare Pages).
2. פתחו את הקישור בכרום בטלפון.
3. אשרו הרשאת מצלמה.
4. בתפריט כרום → **"הוספה למסך הבית"** כדי להתקין כאפליקציה.

> אם אין מצלמה או שלא אושרה הרשאה — המשחק עדיין עובד על רקע צבעוני מונפש.

## 🌐 פרסום ל-GitHub Pages (חינמי, עם HTTPS)
ב-Settings → Pages של הריפו, בחרו את הbranch ואת תיקיית השורש (`/`).
תוך דקה תקבלו כתובת `https://<user>.github.io/creativity/` שעובדת בטלפון.

## 🗂️ מבנה הפרויקט
```
index.html              # מבנה המסכים וה-HUD
css/style.css           # עיצוב, אנימציות, אפקטים
js/game.js              # כל לוגיקת המשחק (מצלמה, ספאון, ניקוד, לוח תוצאות)
manifest.webmanifest    # הגדרות PWA
sw.js                   # Service Worker לאופליין
icons/                  # אייקוני האפליקציה (PNG)
tools/make_icons.py     # סקריפט שמייצר את האייקונים (ללא תלויות)
```

## 🎮 חוקי הניקוד
| אובייקט | נקודות |
|--------|--------|
| יצור 👾 | 10 |
| מטבע 🪙 | 5 |

## 🔧 התאמה אישית
רוב ההגדרות נמצאות בראש `js/game.js` (קבועים מסומנים):
משך סבב, קצב ספאון, סוגי האובייקטים והמשקלים שלהם, נקודות לשלב ועוד.
